## 📐 サイズとスペーシング

### アイコンサイズ
```
小: w-3 h-3  (12px)  ← サイドバーの再生アイコン
中: w-4 h-4  (16px)  ← プレイリストアイコン
大: w-5 h-5  (20px)  ← メニューボタン
特大: w-24 h-24 (96px) ← 共有ページのアイコン
```

### スペーシング
```
space-x-2  (8px)   ← 小さい間隔
space-x-3  (12px)  ← 通常の間隔
space-x-4  (16px)  ← 大きい間隔
space-x-8  (32px)  ← 特大の間隔
```

### パディング
```
p-2  (8px)   ← ボタン
p-4  (16px)  ← カード
p-6  (24px)  ← セクション
p-8  (32px)  ← ページ
```

## 🎬 使用例のシナリオ

### シナリオ1: 友人とプレイリストを共有

```
1. プレイリストを作成
   [+] → "Summer Hits 2026"

2. 曲を追加
   [曲] → [...] → "プレイリストに追加"

3. 共有リンクをコピー
   [プレイリスト] → [...] → "共有リンクをコピー"

4. 友人に送信
   LINE/Discord/Email → リンクを送信

5. 友人がインポート
   [リンクを開く] → [ライブラリに追加]

6. 完了！
   友人のライブラリに追加される
```

### シナリオ2: 全画面プレイヤーで音楽を楽しむ

```
1. 曲を再生
   [曲] → クリック

2. 全画面プレイヤーを開く
   [アートワーク] → クリック

3. 操作
   [▶️] → 再生/一時停止
   [⏮️] → 前の曲
   [⏭️] → 次の曲
   [🔊] → 音量調整

4. 閉じる
   [×] → クリック
```
